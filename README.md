# Kotlin Demo Project

Проект для демонстрации сборки Котлина с помощью Gradle:

* Многомодульный проект Gradle.
* Fat jar.
* Тесты.

## Запуск тестов

```
./gradlew test
```

## Построение fat jar

```
./gradlew jar
```

Запускаемый JAR файл будет создан в папке `build/libs`.
